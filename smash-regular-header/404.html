ERREUR 404: pages ou fichiers introuvables

Les raisons peuvent être les suivantes :

  • La page ou le fichier que vous souhaitez atteindre est provisoirement inaccessible.
  • La page ou le fichier que vous souhaitez atteindre à été rénommé ou n'existe plus.
  • Vous avez cliqué sur un vieux marque-page.
  • Vous avez mal saisi l'URL.
